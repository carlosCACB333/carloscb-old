Este algoritmo funciona bién con datos complejos y de tamaño pequeno o medianos. Se puede aplicar de diferentes formas en funcón de los datos

A comparación de la algoritmos de regresión logística, el SVM genera genera la función de hipótesis lo más lejos posible de los datos de entrenamiento. Esto se conoce como **Large Margin Classification**.

![Large Margin Classification](/note/svm.png)

- **Datos linealmente separables:** los datos se pueden separar con una linea recta.
  - **Hard Margin Classification** (clasificación con márgenes duros)
  - **Soft Margin Classification** (clasificación con márgenes blandos)
- **Datos no linealmente separables:** los datos no se pueden separar con una linea recta.
  - **Gaussian RBF Kernel** (kernel gaussiano RBF)
  - **Polynomial Kernel** (kernel polinomial)
