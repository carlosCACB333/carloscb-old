## Array.prototype.at

El método `Array.prototype.at` nos permite acceder a un elemento de un array por su índice. Este método es similar al método `Array.prototype.slice` pero en lugar de devolver un array con los elementos seleccionados, devuelve el elemento seleccionado.

```js
const array = [1, 2, 3, 4, 5];

array.at(0); // 1
array.at(2); // 3
array.at(-1); // 5
```

## Top level await

El top level await nos permite usar la palabra clave `await` en el scope global. Esto nos permite escribir código asíncrono en el scope global.

```js
const response = await fetch("https://jsonplaceholder.typicode.com/todos/1");
const data = await response.json();

console.log(data);
```
