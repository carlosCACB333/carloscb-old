## La nube

La nube son un conjunto de soluciones basadado en internet que permiten a las empresas y a los individuos utilizar servicios de computación, almacenamiento y otros recursos de TI de forma remota, sin tener que instalar y mantener software o hardware en sus propios equipos. Estos son algunos servicios en la nube:

- Amazon Web Services (AWS)
- Microsoft Azure
- Google Cloud Platform
- IBM Cloud
- Oracle Cloud

## Beneficios de la nube

- Reducción :
  - Costos de hardware
  - Costo operacional
  - Costo de implementación
- Incremento :
  - Escalabilidad
  - Disponibilidad
  - Seguridad
  - Rendimiento
  - Capacidad

## Servicios de AWS

### Servicios de computación

Los servicios de computación son servicios que permiten ejecutar aplicaciones en la nube. Estos servicios incluyen:

- **EC2** : Amazon Elastic Compute Cloud (Amazon EC2) es un servicio web que proporciona capacidad de cómputo en la nube segura y redimensionable. Puede utilizar Amazon EC2 para lanzar instancias de máquinas virtuales (VM) en la nube para ejecutar aplicaciones en la nube. Puede utilizar Amazon EC2 para ejecutar aplicaciones individuales en la nube o como parte de un entorno de computación en la nube más amplio.

- **Elastic Beanstalk** : Amazon Elastic Beanstalk es un servicio de computación en la nube que facilita la implementación, la escala y la administración de aplicaciones y servicios web en AWS sin tener que preocuparse por la infraestructura subyacente. Puede utilizar Elastic Beanstalk para implementar aplicaciones de software preempaquetadas de terceros conocidas como aplicaciones de software listas para usar (USAs), o puede implementar aplicaciones personalizadas que se ejecutan en Amazon EC2, Amazon EC2 Container Service, Amazon EC2 Container Registry o AWS Lambda.

- **Lambda** : AWS Lambda es un servicio de computación en la nube que ejecuta código en respuesta a eventos y automatiza tareas de administración de la infraestructura. Puede usar Lambda para ejecutar código en respuesta a eventos de Amazon S3, Amazon DynamoDB, Amazon Kinesis, Amazon CloudWatch, Amazon Simple Notification Service (Amazon SNS), Amazon Simple Queue Service (Amazon SQS), AWS CodeCommit o AWS CloudFormation. También puede usar Lambda para ejecutar código en respuesta a eventos de terceros a través de Amazon API Gateway o AWS CloudTrail. Puede ejecutar código en respuesta a eventos de tiempo programados mediante AWS CloudWatch Events.

### Almacenamiento

Los servicios de almacenamiento son servicios que permiten almacenar y recuperar datos en la nube. Estos servicios incluyen:

- **S3** : Amazon Simple Storage Service (Amazon S3) es un servicio de almacenamiento de objetos en la nube que ofrece una amplia gama de características de almacenamiento de objetos. Puede usar Amazon S3 para almacenar y recuperar cualquier cantidad de datos en cualquier momento, desde cualquier lugar en la web. Puede almacenar datos de cualquier tipo, incluidos archivos de texto, imágenes, audio y video, así como aplicaciones y sitios web.

- **Elastic File System** : Amazon Elastic File System (Amazon EFS) es un servicio de almacenamiento de archivos en la nube simple y rápido que puede usar para implementar sistemas de archivos de red en AWS. Puede usar Amazon EFS para crear sistemas de archivos de red escalables y altamente disponibles que se pueden montar en múltiples instancias de Amazon EC2 en varias zonas de disponibilidad.

- **Glacier** : Amazon Glacier es un servicio de almacenamiento de datos en la nube que ofrece una duración de conservación de datos de hasta 10 años a un costo extremadamente bajo. Puede usar Amazon Glacier para almacenar datos de cualquier tipo, incluidos archivos de texto, imágenes, audio y video, así como aplicaciones y sitios web.

- **Storage Gateway** : AWS Storage Gateway es un servicio de almacenamiento híbrido que conecta su entorno local con el almacenamiento en la nube de AWS para proporcionar una solución de almacenamiento de datos de bajo costo y de alta disponibilidad. AWS Storage Gateway le permite almacenar datos en el almacenamiento en la nube de AWS de forma transparente, mientras que los datos locales permanecen disponibles para las aplicaciones.

### Bases de datos

Los servicios de bases de datos son servicios que permiten almacenar y recuperar datos en la nube. Estos servicios incluyen:

- **RDS** : Amazon Relational Database Service (Amazon RDS) es un servicio web que facilita la configuración, la operación y la escalabilidad de una base de datos relacional en la nube. Puede utilizar Amazon RDS para implementar bases de datos de MySQL, MariaDB, PostgreSQL, Oracle, Microsoft SQL Server, Amazon Aurora, o bases de datos de Amazon Aurora MySQL compatible con MySQL 5.6 o 5.7. Puede utilizar Amazon RDS para implementar bases de datos en la nube en solo unos minutos con el uso de una interfaz web simple que no requiere conocimientos previos de administración de bases de datos.

- **DynamoDB** : Amazon DynamoDB es un servicio de base de datos NoSQL de alto rendimiento y bajo costo que le permite ejecutar aplicaciones sin servidor que admiten una amplia variedad de esquemas de datos. Puede utilizar Amazon DynamoDB para almacenar y consultar datos de cualquier tipo, incluidos documentos y archivos de audio, video, imágenes y aplicaciones.

- **Redshift** : Amazon Redshift es un servicio de análisis de datos en la nube que le permite ejecutar consultas de análisis de datos de gran velocidad y complejidad en grandes volúmenes de datos de forma rápida y económica. Puede utilizar Amazon Redshift para ejecutar consultas de análisis de datos complejos en datos de transacciones, datos de log, datos de redes sociales, datos de comercio electrónico, datos de marketing y datos de IoT.

- **Elasticache** : Amazon ElastiCache es un servicio de memoria en la nube que facilita la implementación, la operación y la escalabilidad de un sistema de caché en la nube. Puede utilizar Amazon ElastiCache para implementar sistemas de caché de memoria en la nube en solo unos minutos con el uso de una interfaz web simple que no requiere conocimientos previos de administración de sistemas de caché.

### Migración y transferencia

Los servicios de migración y transferencia son servicios que permiten migrar y transferir datos en la nube. Estos servicios incluyen:

- **Migration Hub** : AWS Migration Hub es un servicio de AWS que le ayuda a administrar sus migraciones de aplicaciones y datos a AWS. AWS Migration Hub le permite realizar un seguimiento de sus migraciones de aplicaciones y datos a AWS, y le ayuda a administrar las tareas de migración y los recursos de AWS que se utilizan para migrar a AWS.

- **Application Discovery Service** : AWS Application Discovery Service es un servicio de AWS que le ayuda a realizar un inventario de sus aplicaciones y sus dependencias de infraestructura. AWS Application Discovery Service le permite realizar un inventario de sus aplicaciones y sus dependencias de infraestructura, y le ayuda a migrar sus aplicaciones a AWS.

- **Database Migration Service** : AWS Database Migration Service es un servicio de AWS que le ayuda a migrar sus bases de datos a AWS. AWS Database Migration Service le permite migrar sus bases de datos a AWS.

- **Server Migration Service** : AWS Server Migration Service es un servicio de AWS que le ayuda a migrar sus servidores virtuales a AWS. AWS Server Migration Service le permite migrar sus servidores virtuales a AWS.

- **Transfer family** : AWS Transfer Family es un servicio de AWS que le ayuda a transferir archivos a AWS. AWS Transfer Family le permite transferir archivos a AWS.

- **Snowball** : AWS Snowball es un servicio de AWS que le ayuda a transferir grandes volúmenes de datos a AWS. AWS Snowball le permite transferir grandes volúmenes de datos a AWS.

### Redes y entrega de contenido

Los servicios de redes y entrega de contenido son servicios que permiten implementar y administrar redes y entregar contenido en la nube. Estos servicios incluyen:

- **VPC** : Amazon Virtual Private Cloud (Amazon VPC) le permite provisionar una porción lógica de la infraestructura de AWS en la nube, de forma similar a una red privada. Puede utilizar Amazon VPC para implementar recursos en la nube de forma segura y controlar el tráfico de red entrante y saliente.

- **CloudFront** : Amazon CloudFront es un servicio de entrega de contenido en la nube que acelera la entrega de contenido estático y dinámico a los usuarios finales. Puede utilizar Amazon CloudFront para entregar contenido a los usuarios finales con baja latencia, alta velocidad y seguridad.

- **Route 53** : Amazon Route 53 es un servicio de DNS en la nube que le permite conectar usuarios finales a recursos de AWS, como sitios web y aplicaciones. Puede utilizar Amazon Route 53 para registrar dominios de nivel superior y subdominios, y para configurar DNS para que los usuarios finales puedan acceder a sus recursos de AWS.

- **Api Gateway** : Amazon API Gateway es un servicio de AWS que le ayuda a crear, publicar, mantener, monitorear y proteger APIs en AWS. Puede utilizar Amazon API Gateway para crear, publicar, mantener, monitorear y proteger APIs en AWS.

- **Direct Connect** : AWS Direct Connect es un servicio de AWS que le ayuda a establecer una conexión dedicada privada entre su red local y AWS. Puede utilizar AWS Direct Connect para establecer una conexión dedicada privada entre su red local y AWS.

- **App Mesh** : AWS App Mesh es un servicio de AWS que le ayuda a administrar y monitorear aplicaciones distribuidas. Puede utilizar AWS App Mesh para administrar y monitorear aplicaciones distribuidas.

### Administración y Gobierno

Los servicios de administración y gobierno son servicios que permiten administrar y controlar los recursos de AWS. Estos servicios incluyen:

- **CloudWatch** : Amazon CloudWatch es un servicio de monitoreo en la nube que le permite monitorear sus recursos de AWS y las aplicaciones que ejecuta en AWS. Puede utilizar Amazon CloudWatch para monitorear sus recursos de AWS y las aplicaciones que ejecuta en AWS.

- **Auto Scaling** : Amazon EC2 Auto Scaling es un servicio de AWS que le ayuda a ajustar automáticamente el número de instancias EC2 para optimizar el costo y la disponibilidad de sus aplicaciones. Puede utilizar Amazon EC2 Auto Scaling para ajustar automáticamente el número de instancias EC2 para optimizar el costo y la disponibilidad de sus aplicaciones.

- **CloudFormation** : AWS CloudFormation es un servicio de AWS que le ayuda a modelar y configurar una colección de recursos de AWS, y a implementar y administrar estos recursos de forma predecible y repetible. Puede utilizar AWS CloudFormation para modelar y configurar una colección de recursos de AWS, y para implementar y administrar estos recursos de forma predecible y repetible.

- **System Manager** : AWS Systems Manager es un servicio de AWS que le ayuda a administrar sus recursos de AWS. Puede utilizar AWS Systems Manager para administrar sus recursos de AWS.

### Servicios de multimedia

Los servicios de multimedia son servicios que permiten almacenar, procesar y entregar contenido multimedia en la nube. Estos servicios incluyen:

- **Elastic Transcoder** : Amazon Elastic Transcoder es un servicio de AWS que le ayuda a convertir contenido multimedia.

- **MediaConvert** : AWS Elemental MediaConvert es un servicio de AWS que le ayuda a convertir contenido multimedia.

- **MediaLive** : AWS Elemental MediaLive es un servicio de AWS que le ayuda a transmitir contenido multimedia en vivo.

### Machine Learning

Los servicios de machine learning son servicios que permiten implementar y administrar modelos de machine learning en la nube. Estos servicios incluyen:

- **SageMaker** : Amazon SageMaker es un servicio de AWS que le ayuda a desarrollar, entrenar e implementar modelos de machine learning.

- **Rekognition** : Amazon Rekognition es un servicio de AWS que le ayuda a identificar objetos, personas, texto, escenas y actividades en imágenes y videos.

- **Lex** : Amazon Lex es un servicio de AWS que le ayuda a construir conversaciones interactivas con sus usuarios.

### Seguridad, identidad y conformidad

- **IAM** : AWS Identity and Access Management (IAM) es un servicio de AWS que le ayuda a administrar el acceso a los recursos de AWS. Puede utilizar IAM para administrar el acceso a los recursos de AWS.

- **Inspector** : Amazon Inspector es un servicio de AWS que le ayuda a auditar la configuración de seguridad de sus instancias EC2. Puede utilizar Amazon Inspector para auditar la configuración de seguridad de sus instancias EC2.

- **CloudHSM** : AWS CloudHSM es un servicio de AWS que le ayuda a proteger sus claves de cifrado en un hardware de seguridad de nivel empresarial.

### Administración de costo

- **Cost Explorer** : AWS Cost Explorer es un servicio de AWS que le ayuda a administrar el costo de sus recursos de AWS. Puede utilizar AWS Cost Explorer para administrar el costo de sus recursos de AWS.

- **Budgets** : AWS Budgets es un servicio de AWS que le ayuda a administrar el presupuesto de sus recursos de AWS. Puede utilizar AWS Budgets para administrar el presupuesto de sus recursos de AWS.

### Móvil

- **Mobile Hub** : AWS Mobile Hub es un servicio de AWS que le ayuda a crear, administrar y monitorear aplicaciones móviles en AWS. Puede utilizar AWS Mobile Hub para crear, administrar y monitorear aplicaciones móviles en AWS.

- **Device Farm** : AWS Device Farm es un servicio de AWS que le ayuda a probar aplicaciones móviles en dispositivos reales.
