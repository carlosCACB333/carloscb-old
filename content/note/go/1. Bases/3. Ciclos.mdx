## For

El ciclo for es la única estructura de control que existe en Go, y tiene varias formas de uso, la primera es como un for tradicional, en el cual se inicializa una variable, se evalúa una condición y se ejecuta el bloque de código mientras se cumpla la condición, y al final se ejecuta una instrucción.

```go
package main

import "fmt"

func main() {
  for i := 0; i < 10; i++ {
    fmt.Println(i)
  }
}
```

## For while

El ciclo for también puede ser usado como un ciclo while, en el cual se evalúa una condición y se ejecuta el bloque de código mientras se cumpla la condición.

```go
package main

import "fmt"

func main() {
  i := 0
  for i < 10 {
    fmt.Println(i)
    i++
  }
}
```

## For forever

El ciclo for también puede ser usado como un ciclo infinito, en el cual no se evalúa ninguna condición, y se ejecuta el bloque de código hasta que se rompa el ciclo.

```go
package main

import "fmt"

func main() {
  for {
    fmt.Println("Hola")
  }
}
```

## break

La instrucción break se utiliza para romper un ciclo, y se puede usar en cualquier ciclo, ya sea for, while o forever.

```go
package main

import "fmt"

func main() {
  for {
    fmt.Println("Hola")
    break
  }
}
```

## continue

La instrucción continue se utiliza para saltar a la siguiente iteración del ciclo, y se puede usar en cualquier ciclo, ya sea for, while o forever.

```go
package main

import "fmt"

func main() {
  for i := 0; i < 10; i++ {
    if i == 5 {
      continue
    }
    fmt.Println(i)
  }
}
```
