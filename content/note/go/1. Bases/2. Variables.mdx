Go es un lenguaje de programación que tiene un sistema de tipos estático. Esto significa que las variables deben ser declaradas antes de ser usadas y que el tipo de dato de una variable no puede cambiar durante la ejecución del programa. En este artículo aprenderás a declarar, asignar y los distintos tipos de datos que existen en Go.

## Operador de asignación corta `:=`

El operador de asignación corta `:=` es una forma abreviada de declarar e inicializar una variable. El operador `:=` es equivalente a la declaración `var` seguida de una asignación. Estas tienen ciertas reglas:

- El operador `:=` solo puede ser usado dentro de una función.
- La variable debe ser nueva, no puede ser una variable ya existente.
- El tipo de la variable se infiere del valor que se le asigna.

```go
func main() {
    a := 10
    b := "golang"
    fmt.Println(a, b, c, d)
}
```

## Declaración y asignación de variables

Las variables se declaran con la palabra reservada `var` seguida del nombre de la variable y el tipo de dato que contendrá. La declaración de variables se puede hacer de forma individual o en grupo.

- Con `var` podemos tener variables globales o locales.
- Las variables declaradas sin un valor inicial se inicializan con el valor cero de su tipo de dato.

```go
var a int
var b string = "golang"
var (
    c int
    d string
)

func main() {
    fmt.Println(a, b, c, d)
}

```

## constantes

Las constantes son valores que no cambian durante la ejecución del programa. Se declaran con la palabra reservada `const` seguida del nombre de la constante y el valor que contendrá.

```go
import "fmt"
const pi = 3.1416

func main() {
    fmt.Println(pi)
}

```

## Cero values

Go tiene un valor cero para cada tipo de dato. Este valor se asigna a las variables cuando no se les asigna un valor explícitamente.

| Tipo de dato                                  | Valor cero |
| --------------------------------------------- | ---------- |
| bool                                          | false      |
| int                                           | 0          |
| float                                         | 0.0        |
| string                                        | ""         |
| pointer, func, interface, slice, channel, map | nil        |

## Tipos definidos por el usuario

Los tipos definidos por el usuario son tipos que se crean a partir de otros tipos.

```go
type Celsius float64

func main() {
    var c Celsius
    fmt.Printf("%T\n", c)
    // Output: main.Celsius
}

```
