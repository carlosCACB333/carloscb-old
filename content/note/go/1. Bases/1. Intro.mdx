## Características

**Go es eficiente complilación, eficiente ejecución y fácil programación.**

- **Performance:** Aplicaciones de alto rendimiento.
- **Núcleos múltiples:** Soporta de múltiples núcleos.
- **concurrente:** Soporta la programación concurrente.
- **compilado:** En distintas plataformas.
- **Network:** Para la comunicación entre procesos.
- **Sintaxis simple:** Sintaxis simple y legible.
- **Librería estándar poderosa:** Librería estándar poderosa y extensible.
- **Garbage collector:** Recolector de basura.
- **Portable:** Compilado para distintas plataformas.
- **Apoyado por Google:** Apoyado por Google.
- **Open source:** Código abierto.

## Terminología

- **Palabras claves:** Son palabras reservadas por el lenguaje, no se pueden usar como identificadores.
- **Operador:** Es un símbolo que indica una operación.
- **Operando:** Es el valor sobre el que se realiza la operación.
- **declaración:** Es el elemento más pequeño que expresa la acción que se tiene que realizar la computadora.
- **Expresión:** Es una combinación de valores, variables, operadores y llamadas a funciones que se evalúa para producir un valor.
